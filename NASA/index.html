First Time User

To log in using NASA Access Launchpad you will need to create a profile. Creating your profile will allow you to set a password and security answers which will be used should you ever forget your password. Once it is completed your account will be enabled for log in.

To setup your profile, click the Create My Profile button below which will take you to User Self Service in order to complete the process.

If you have already created your profile, you can close this dialog box and log in.
